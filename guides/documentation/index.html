Install Demo Quickstart

The Quickstart Package consists on a complete Opencart! + Template + Various Extensions + Sample Content, excellent for beginner users to explore back-end settings and sample content. Installing the Quickstart is just like installing Opencart!, but you will have the full demo of the theme on your server. Once your environment is ready, follow this quick guide:

  • Localhost only: create a new folder in the htdocs folder for Xampp (www folder for Wamp). Example: mywebsite
  • Unpack all files and folders of the Template Package file and open the Quickstart folder.
  • Copy the content (all files and folders) of the Quickstart folder into this newly created folder (htdocs folder for Xampp / www folder for Wamp) or upload them into your sub-domain folder on your live site.
  • Open your favorite browser and type in the address bar localhost/mywebsite for localhost or sub-domain.my-live-site.com for your live site sub-domain, then hit Enter.
  • Watch this video tutorial on how to install Opencart

 

Install Theme

Before you install this theme:

1. Read the complete instruction
2. Make sure you have installed and running the correct version of OpenCart
3. It is a good practice to backup your system

We recommend to use this theme on a new, clean installation or existing shops without custom core modifications.

Install Module

1. Please unzip package Modules and upload the folder admin,catalog ...  from the Module package to the root directory of your Opencart software using an FTP client.

2. Go to Admin > Extensions > Modules . Find and install it or uninstall
